Benjamin Lupton's Website, built with DocPad Getting Started

  1. Install DocPad
  2. Clone and run the server
     git clone git://github.com/balupton/balupton.docpad.git
 cd balupton.docpad
 npm install
 docpad run
  3. Open http://localhost:9778/
  4. Start hacking away by modifying the src directory
